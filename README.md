# Forkify 🍴

A simple recipe app! Search for your favorite food and also store your grandmother's delicious recipes. 

Enjoy! 🧇 🥩

## ~~Snack~~ Stack
- Architecture: Object-oriented programming (OOP), Model–view–controller (MVC)
- Backend: Node
- Bundler: Parcel
- Frontend: VanillaJS
- Hosting: Cloudflare Pages
